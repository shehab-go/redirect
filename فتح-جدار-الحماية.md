# ğŸ”“ Opening Firewall for Access from Other Devices

## Method 1: Using PowerShell Script (Recommended)

1. Right-click on `open-firewall.ps1` file
2. Select **"Run as Administrator"**
3. If security prompt appears, click **"Yes"**

## Method 2: Open Firewall Manually

### Windows Steps:

1. Press `Win + R`
2. Type: `wf.msc` and press Enter
3. From left menu, select **"Inbound Rules"**
4. Click **"New Rule..."** from right menu
5. Select **"Port"** then **Next**
6. Select **"TCP"** and enter port: **8000** then **Next**
7. Select **"Allow the connection"** then **Next**
8. Check all options (Domain, Private, Public) then **Next**
9. Enter rule name: **"Python Server Port 8000"** then **Finish**

### Quick Method (Command Prompt as Administrator):

```cmd
netsh advfirewall firewall add rule name="Python Server Port 8000" dir=in action=allow protocol=TCP localport=8000
```
